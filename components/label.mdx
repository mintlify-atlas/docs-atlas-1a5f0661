title: Label

## Installation

```bash
npm install @tamagui/label
```

## Import

```tsx
import { Label } from '@tamagui/label'
```

## Usage

### Basic Example

```tsx
import { Label } from '@tamagui/label'
import { Input } from '@tamagui/input'
import { YStack } from '@tamagui/stacks'

function Demo() {
  return (
    <YStack space>
      <Label htmlFor="email">Email Address</Label>
      <Input id="email" type="email" />
    </YStack>
  )
}
```

### Wrapping Input

```tsx
import { Label } from '@tamagui/label'
import { Input } from '@tamagui/input'

function Demo() {
  return (
    <Label>
      Username
      <Input />
    </Label>
  )
}
```

## API Reference

### Label Props

<ParamField path="htmlFor" type="string">
  ID of the form control this label is associated with. Sets `aria-labelledby` on the target element.
</ParamField>

<ParamField path="size" type="SizeTokens">
  Size of the label text. Matches button sizing for consistent form layouts.
</ParamField>

<ParamField path="id" type="string">
  ID attribute for the label element.
</ParamField>

<ParamField path="onPress" type="(event) => void">
  Callback when label is pressed/clicked.
</ParamField>

<ParamField path="children" type="ReactNode">
  Label text or child elements.
</ParamField>

## Theming

Label responds to the following theme tokens:

- `$color` - Text color
- `$colorPress` - Text color when pressed

```tsx
import { Label } from '@tamagui/label'

function Demo() {
  return (
    <Label theme="blue" htmlFor="input">
      Themed Label
    </Label>
  )
}
```

## Accessibility

- Renders as a semantic `<label>` element
- Automatically associates with form controls via `htmlFor`
- Sets `aria-labelledby` on the associated control
- Clicking label focuses the associated control
- Supports wrapping controls for implicit association
- Prevents text selection on double-click
- `cursor: default` for proper cursor styling
- Works with screen readers to announce form controls

## Behavior

### With `htmlFor`

When using `htmlFor`, the label must reference a control by ID:

```tsx
<Label htmlFor="email">Email</Label>
<Input id="email" />
```

The label will:
- Set `aria-labelledby` on the input
- Focus the input when clicked
- Work with all form controls (Input, Checkbox, Switch, etc.)

### Wrapping Controls

Labels can wrap controls for implicit association:

```tsx
<Label>
  Email
  <Input />
</Label>
```

In this case:
- No `htmlFor` or `id` needed
- Clicking the label text focuses the input
- Clicking the input itself doesn't trigger label click handler

### Native Platform

On React Native:
- Uses `focusFocusable()` to programmatically focus controls
- Requires `htmlFor` to match the control's `id`
- Focusable controls must be registered (automatically done by Tamagui form components)

## Notes

- Size automatically matches button sizing for consistent form layouts
- Line height calculated to align with button heights
- Default cursor is `default`, not `pointer` (follows native label behavior)
- Prevents text selection when double-clicking
- `userSelect: none` prevents dragging selection
- `display: flex` and `alignItems: center` for easy icon inclusion
